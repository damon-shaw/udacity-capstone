Welcome to the Virginia PlateSearch Tool!

This is a tool I built in my free time to make searching for a custom license plate easier. As of August 2020, the Virginia DMV website was slow and hard to work with.

This tool started as a simple script that would submit the completed form to the DMV site. I was looking for a full seven letter word that was available in its normal spelling (MYPLATE vs MYPL4T3). While the script was fast at putting together the request, the Virginia DMV site was still slow. To resolve this, I created my own database, and automated requests for thousands of words. With my own database I could query for specific plates, query for plates that included words, or view random available plates.

I didn't want the database to only hold the plates from the wordlist the automated tool referenced, though, so this site is capable of searching for new plates. Also, as a proxy to the actual Virginia DMV database, this site is not the authority on what's available at what isn't. As such, the ability to check a plate's availability was a critical feature.
